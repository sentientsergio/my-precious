# Smoke

## Goal
Given a short list of topic names (in the run context), write a one-paragraph
"did you know" brief for each: one specific, checkable fact plus why it's
interesting. Keep each brief under 100 words.

## Constraints
- Don't invent facts. If you're not confident in one, say so instead of
  guessing.
- One file per topic, named after the topic, plus a single index file
  listing all of them with a one-line summary each.
- Keep the whole run small — a handful of topics, not a research project.
